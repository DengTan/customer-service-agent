# 坐席工作台视觉优化设计方案

**日期**: 2026-07-10
**状态**: 已批准

## 目标

- 视觉层级更清晰，区分主次内容，突出当前服务对话
- 界面更现代、专业，与项目其他页面风格统一

## 改进范围

### 1. 左侧排队列表优化 (queue-panel.tsx)

#### 排队/服务中切换
- 从两个独立区块改为 Tab 切换（类似浏览器标签页）
- Tab 样式：紧凑胶囊式，选中时底部有主题色指示条
- 减少视觉层级，更加紧凑高效

#### 列表项样式
- 卡片式设计：圆角 8px，柔和背景色
- 选中状态：左侧 3px 主题色边框 + 浅色背景
- 悬停状态：背景色微变
- 服务中绿点更醒目
- 优先级徽章：urgent 红色，其他使用柔和蓝灰色

#### 间距统一
- 统一 padding 为 12px
- 栏间分隔改为 1px 浅色线

### 2. 右侧客户信息优化 (customer-info-panel.tsx)

#### 卡片布局
- 信息分组更清晰：
  - **基本信息**：头像 + 姓名 + 来源平台
  - **问题摘要**（如有）：摘要内容
  - **转人工原因**（如有）：原因内容
  - **快捷操作**：操作按钮组

#### 操作按钮
- 统一为 icon + 文字的紧凑按钮
- 圆角统一，hover 效果一致
- 结束对话按钮使用淡红色警示

#### 间距
- 区块间距增加，信息更有呼吸感
- padding 统一为 16px

### 3. 整体布局 (workspace-page.tsx)

#### 栏间分隔
- 改为 1px 浅色分隔线（而非粗边框）
- 三栏视觉更融合，不被切割感

#### Header 统计区域
- 统计数字使用较大字号
- 数值颜色使用主题强调色
- 状态切换按钮组更紧凑

## 组件改动

| 文件 | 改动内容 |
|------|---------|
| `queue-panel.tsx` | Tab 切换、卡片样式优化 |
| `customer-info-panel.tsx` | 卡片分组、操作按钮样式 |
| `workspace-page.tsx` | 三栏布局间距调整 |
